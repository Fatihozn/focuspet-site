# focuspet-site

Public site for the FocusPet iOS app — landing page, privacy policy, and support page.

Hosted via GitHub Pages.

## Pages
- `index.html` — landing
- `privacy.html` — privacy policy (EN + TR)
- `support.html` — support page (EN + TR)

## Local preview

Open `index.html` directly in a browser, or:

```sh
python3 -m http.server 8000
# then visit http://localhost:8000
```

## Deploy

Push to `main`. GitHub Pages serves from the repository root.
